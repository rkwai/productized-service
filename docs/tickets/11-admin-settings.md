# Ticket: Admin / Settings improvements

## Goal
Enhance admin configurability with clearer insights, structured editing, and governance workflows.

## Scope
- Admin KPIs, configuration editors, audit logs, and data integrations.
- Approval workflows and scheduled data refresh controls.

## Tasks
### Stream A — Insights & Metrics
- Add config diffing and validation reports.
- Add data pipeline latency/quality KPIs.
- Add audit summary trends.

### Stream B — Visuals & UX
- Add structured editors for config sections.
- Add search/filter/pagination in audit logs.
- Add data integration status dashboard.

### Stream C — Workflow & Actions
- Add approval workflow for config changes.
- Add scheduled refresh and pipeline run controls.
- Add export/report automation for audit data.

## Acceptance Criteria
- Config changes show diffing and validation results.
- Audit logs support search, filters, and pagination.
- Data integration status is visible.
- Approvals and scheduled refresh actions are available.
