# Ticket: Ontology Explorer improvements

## Goal
Make ontology objects discoverable with metrics, search, and relationship visualization.

## Scope
- Object type selector, search, and object type cards.
- Relationship insights and schema change workflow.

## Tasks
### Stream A — Insights & Metrics
- Add object usage metrics and data freshness stats.
- Add relationship density metrics.
- Add schema change impact analysis.

### Stream B — Visuals & UX
- Enable search and filters.
- Add relationship graph view.
- Add “Used in modules” links to pages.

### Stream C — Workflow & Actions
- Add schema change request flow.
- Add validation checks for new fields.

## Acceptance Criteria
- Object cards show usage and freshness metrics.
- Search and filters are functional.
- Relationship graph is accessible from the page.
- Schema change requests include validation checks.
