# Ticket: Action Center / Inbox improvements

## Goal
Improve queue prioritization, visibility, and bulk workflow execution.

## Scope
- Action queue KPIs, action lists, and action log history.
- Bulk execution, approvals, and escalation routing.

## Tasks
### Stream A — Insights & Metrics
- Add backlog aging and SLA breach metrics.
- Add priority scoring for action queues.
- Add owner workload distribution.

### Stream B — Visuals & UX
- Add queue tabs and advanced filters.
- Add action preview pane with context.
- Add status chips and timeline indicators.

### Stream C — Workflow & Actions
- Add bulk action execution and approvals.
- Add escalation routing rules.
- Add automation triggers from other pages.

## Acceptance Criteria
- KPIs display backlog aging and SLA breaches.
- Queues support tabs, filters, and previews.
- Actions can be bulk executed with approvals.
- Escalation routing is configurable and functional.
