# Ticket: Portfolio (Accounts) improvements

## Goal
Provide deeper portfolio insights, richer segmentation visuals, and operational workflows for account management.

## Scope
- Portfolio KPIs, charts, and accounts table.
- Bulk account actions and exports.

## Tasks
### Stream A — Insights & Metrics
- Add cohort metrics by segment/region with deltas.
- Add “data freshness” KPI and missing data flags per account.
- Add churn risk score and LTV-at-risk calculation.

### Stream B — Visuals & UX
- Implement segment/region breakdown charts.
- Add column filters, grouping, and saved table views.
- Add inline health/risk color indicators.

### Stream C — Workflow & Actions
- Add bulk actions (assign owner, create task).
- Add export/report scheduling options.

## Acceptance Criteria
- KPIs include cohort deltas and data freshness.
- Charts display segment and region breakdowns.
- Table supports filtering and grouping with saved views.
- Bulk actions and scheduled export are available from the table.
