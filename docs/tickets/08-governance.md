# Ticket: Governance improvements

## Goal
Add decision health insights, meeting visibility, and action assignment workflows.

## Scope
- Governance KPIs, meetings table, and decisions table.
- Action item assignment and executive updates.

## Tasks
### Stream A — Insights & Metrics
- Add “decision backlog” and action item closure rate KPIs.
- Add sentiment trend by stakeholder group.
- Add meeting impact score.

### Stream B — Visuals & UX
- Add meeting timeline and decision status chips.
- Provide summary cards for pending decisions.
- Add search and filtering for decisions.

### Stream C — Workflow & Actions
- Add “Assign action item” workflow.
- Add “Publish executive update” action.

## Acceptance Criteria
- KPIs include decision backlog and closure rate.
- Meeting timeline and decision chips are visible.
- Decisions table supports search and filtering.
- Action item and executive update workflows are actionable.
