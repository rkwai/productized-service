# Ticket: Value Realization improvements

## Goal
Track outcome performance with confidence scoring, clearer visuals, and proactive follow-ups.

## Scope
- Outcomes KPIs, charts, and outcomes table.
- Outcome updates and linkage to milestones.

## Tasks
### Stream A — Insights & Metrics
- Add confidence score per outcome.
- Add target vs. baseline delta KPI and variance tracking.
- Add stalled KPI detection with thresholds.

### Stream B — Visuals & UX
- Replace charts with trendlines and roll-up view.
- Highlight top/bottom outcomes.
- Add progress indicator with forecasted attainment date.

### Stream C — Workflow & Actions
- Add “Request KPI update” action.
- Link outcomes to milestones and tasks.

## Acceptance Criteria
- Outcomes include confidence and variance metrics.
- Trendlines and roll-ups replace placeholders.
- Top/bottom outcomes are highlighted.
- KPI update requests are actionable from the page.
